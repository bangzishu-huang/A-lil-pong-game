# A-lil-pong-game
Kind of a pong game that I will recreate from absolute scratch. This will be a solo player game and the goal of this game is essentially to catch and bounce the ball back to your opponent (which in this case will be a computer). If you miss a bounce or the ball flies past your paddle you the other side gets a point! This can be played infinitely so the fun will never stop!

This is probably the first game I will be using pygame on, and I had to learn how to utilize a lot of the features. This was a pretty good learning process for me and I was really happy to finish the game!

# Features
While this game is pretty straightforward a couple of the features include:
- Movable player paddle (using your up and down arrow keys)
- Computer controlled paddle that acts as your opponent
- Menu on start up which will give you the option of either continuing where you left off or starting a new game
- Stores your most previous match score so you have the ability to continue later on
